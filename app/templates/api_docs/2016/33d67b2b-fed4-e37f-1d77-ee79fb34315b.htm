bFlag
Type: System Boolean
True to delay the display of the mini-warning dialog until the end of the next transation, false to display them as this transaction is completed.

Return Value

This FailureHandlingOptions object.

Remarks

This controls warnings suitable for the mini-warnings dialog only. If the modal flag is set to true with SetForcedModalHandling(Boolean) then this flag will be ignored.

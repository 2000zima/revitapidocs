Test the type id to make sure it satisfies the following conditions - It is a valid element id - It corresponds to a valid DirectShapeType - The DirectShapeType has the same category assigned. Additionally, this functions tests that the current type id in this DirectShape is invalid. The type id is initialized to invalidElementId by the create functions. Once it is set, it may no longer be changed.

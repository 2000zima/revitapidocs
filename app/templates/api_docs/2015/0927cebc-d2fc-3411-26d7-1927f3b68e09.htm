Members

Member name Description
Visible The geometry object is always visible. This type of geometry is always returned by Geometry Options
Highlight The geometry object is visible, but is affected by highlight or prehighlight special coloration. This type of geometry is returned by Geometry Options if IncludeNonVisibleObjects is true.
Contextual The geometry object is visible in some contexts, but not in others. This type of geometry is always returned by Geometry Options , even when Revit is in a context where the geometry should not be visible, but only if IncludeNonVisibleObjects is true. Examples of geometry of this class include Solids representing insulation and lining for MEP ducts and pipes.
Invisible The geometry object is never visible. This type of geometry is never returned by Geometry Options but may be obtained by other API operations.

Return Value

One or more component repeater objects representing the result pattern of the repeating operation.

Remarks

All elements must be adaptive family instances, have no shape handles, and have at least one placement point hosted on a 1D or 2D repeating reference. The remaining placement points must be either unhosted or hosted on another repeating reference.

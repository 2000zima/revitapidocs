Return Value

True if the referenced Element can currently host Rebar elements, false otherwise.

Remarks

If GetRebarHostData() returns a RebarHostData object, but RebarHostData.IsValidHost() returns false, this means that the element can be made a valid rebar host, generally by setting a property like FLOOR_PARAM_IS_STRUCTURAL or by changing the element's physical material to concrete. On the other hand, if GetRebarHostData() returns a null reference ( Nothing in Visual Basic) , the element cannot be made into a rebar host. This method replaces the Valid property from the 2011 Revit API.

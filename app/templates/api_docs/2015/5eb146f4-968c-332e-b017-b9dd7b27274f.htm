If replacement was successful the new family parameter is returned, otherwise an exception with failure information will be thrown.

Exceptions

Exception Condition
Autodesk.Revit.Exceptions ArgumentNullException Thrown when the input argument-"currentParameter" or "parameterName"-is a null reference ( Nothing in Visual Basic) .

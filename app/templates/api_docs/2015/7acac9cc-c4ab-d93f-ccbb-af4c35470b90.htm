Remarks

When changing the layout rule to MinimumClearSpacing, you must also simultaneously set Spacing, SetLength, BarsOnNormalSide, IncludeFirstBar, and IncludeLastBar properties.

Exceptions

Exception Condition
Autodesk.Revit.Exceptions ArgumentOutOfRangeException The spacing isn't bigger than 0.0. -or- the set length arrayLength isn't acceptable.
Autodesk.Revit.Exceptions DisabledDisciplineException None of the following disciplines is enabled: Structural.

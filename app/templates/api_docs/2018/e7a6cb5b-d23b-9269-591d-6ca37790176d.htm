Return Value

True when the server should be submitting transparent objects for rendering, false otherwise.

Remarks

Opaque and transparent geometry should be submitted for rendering separately. See Autodesk::Revit::DB::DirectContext3D::IDirectContext3DServer::UseInTransparentPass(void).

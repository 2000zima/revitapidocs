Parameters

name
Type: System..::..String
The not empty name for the Load Usage Element to create.

Return Value

If successful and there isn't the Load Usage Element with the same name NewLoadUsage returns an object for the newly created LoadUsage. If such element exist it returns existing element. nullNothingnullptra null reference (Nothing in Visual Basic) is returned if the operation fails.
